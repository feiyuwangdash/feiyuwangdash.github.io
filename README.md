# Personal Website

This is my personal website hosted on GitHub Pages at [feiyuwangdash.github.io](https://feiyuwangdash.github.io).

## Structure

- `index.html`: Main HTML file
- `css/style.css`: Stylesheet
- `js/script.js`: JavaScript for interactivity
- `assets/`: Directory for images and other assets

## Development

To view locally, open `index.html` in a web browser.

To deploy, push changes to the `main` branch of the repository `feiyuwangdash/feiyuwangdash.github.io` on GitHub.

## Customization

Edit the placeholders in `index.html` to add your personal information, projects, and contact details.
